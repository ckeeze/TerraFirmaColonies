Tips and rules:
  I recommend using IntelliJ as your java dev envivorment if you want to contribute java code to the project. 
  Please test your contributions before making a pull request as I will only check for stability (crashes) and not functionality.

  Targeted versions:
  - Minecraft 1.20.1
  - Forge 47.4.18
  - Minecolonies 1.1.1170 (we are using a slightly outdated version for possible War'n Taxes compatibility)
  - Domum Ornamentum 1.0.295
  - Structurize 1.0.800
  - Multi-Piston 1.2.43-RELEASE

  Do not work on features that are marked as assigned or WIP to avoid confusion, if you believe the currently assigned contributor abbandoned the task ask Ckeeze first!
  When you begin work on a feature mark that feature as assigned in the Trello table https://trello.com/b/29ss2UaI/terrafirmacolonies
  When your pull request get's accepted and merged into the main branch you can mark your feature as completed.
  Bugs are also internally tracked at the Trello table, the Issues here are for users to report, not for internal bug tracking.
  Json / Datapack work doesn't need IntelliJ but it's easier to test your work if you have it properly set up.
  If you have questions reach out to Ckeeze on discord: oplosz#3360
  If you are working with us in exchange for monetary compensation and have finance related questions reach out to diptopted: diptopted#4510
  Thank you all for your contributions! - from Ckeeze
